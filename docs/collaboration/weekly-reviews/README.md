# Weekly Reviews

Weekly reviews summarize what changed during a week and whether Codex collaboration improved progress or efficiency.

## File Naming

Use ISO week naming:

```text
YYYY-WW.md
```

Example:

```text
2026-W25.md
```

```markdown

- TBD

- TBD

- TBD
```
